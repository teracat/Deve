# Deve
Welcome to the **.NET DEVEloper Template**! This project serves as a starting point for developers who need to create applications with data access requirements. The template is designed to be flexible and adaptable, allowing you to change data access strategies without needing to overhaul the entire project.

![Diagram](https://raw.githubusercontent.com/teracat/Deve/e320e0385e61bb78b447f2adb70805ee52aaabc3/diagram.png)

## Installation

Check the [Step-by-Step Installation Guide](https://github.com/teracat/Deve/wiki/Installation#step-by-step-installation-guide) in the Wiki section.

In Version 2, you can use new commands to add Modules, Features and Methods. See [New commands in Version 2](https://github.com/teracat/Deve/wiki/Installation#new-commands-in-version-2).

## Clients Samples

Client app samples are provided to show how to use the Core or the Sdk + Api to access the data. More information in [Clients](https://github.com/teracat/Deve/wiki/Clients).

## Deploy

The project includes GitHub workflows for CI (Continuous Integration) and CD (Continuous Delivery). More information about how to automate the deployment in the [Deploy](https://github.com/teracat/Deve/wiki/Deploy) section.

## Diagnostics

Telmetry is added to the Api (also to the MAUI Client sample). More information in [Diagnostics](https://github.com/teracat/Deve/wiki/Diagnostics).
